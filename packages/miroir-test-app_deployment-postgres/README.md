# miroir-test-app_deployment-postgres

Example Postgres application assets for the Miroir Framework.

This package contains the model and data definitions for a simple postgres data model management system, used as an example of as useful application developped with the Miroir Framework.

## Contents

- `assets/postgres_model/`: Entity definitions, reports, endpoints, and other model components
- `assets/postgres_data/`: Sample data instances (entities and entity definitions)
