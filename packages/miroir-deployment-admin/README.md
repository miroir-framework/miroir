# miroir-deployment-admin

Admin deployment assets for the Miroir Framework.

This package contains the admin data model and instance data used by the Miroir Framework for administrative deployments.

## Contents

- **admin_model**: Entity definitions, reports, endpoints, and other model artifacts for the admin application
- **admin_data**: Instance data for admin deployments

## Usage

```typescript
import { 
  entityApplicationForAdmin,
  deployment_Admin,
  // ... other exports
} from 'miroir-deployment-admin';
```
