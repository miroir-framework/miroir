# miroir-test-app_deployment-miroir

Miroir deployment assets for the Miroir Framework.

This package contains the Miroir meta-model and data instances (previously in `miroir-core/src/assets/miroir_model` and `miroir-core/src/assets/miroir_data`).

## Contents

- **miroir_model**: Entity and EntityDefinition instances for the Miroir meta-model
- **miroir_data**: Data instances (Reports, Endpoints, JzodSchemas, Queries, Menus, Transformers, Runners, Tests, etc.)

## Usage

```typescript
import { 
  entityEntity,
  entityDefinitionEntity,
  reportEntityList,
  menuDefaultMiroir,
  // ... other exports
} from 'miroir-test-app_deployment-miroir';
```
